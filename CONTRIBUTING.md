# Contributing

This repository is a **read-only mirror** of the `packages/sdk` directory in [botchaxyz/botcha](https://github.com/botchaxyz/botcha). It is synced automatically on every push to `main`.

**Do not open pull requests here** -- they will be closed automatically.

To contribute, please file issues or open PRs against the monorepo: [botchaxyz/botcha](https://github.com/botchaxyz/botcha).
